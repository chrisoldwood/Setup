Setup v1.2
==========


1: Installation
---------------

Run setup.exe.

or

Copy all the files in the archive to a folder.


2: Uninstallation
-----------------

Delete all the files from the archive.


3: Manual
---------

Setup.exe requires a script file called setup.ini, which must reside in the same
folder as the executable and any files you are installing. An example one is
provided, which has comments to tell you what to set. Alternatively, you can use
the GUI tool called MkSetup.exe to generate the script.

NB: It has no Uninstall support at present.
